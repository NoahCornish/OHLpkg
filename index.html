What is OHLpkg?

`OHLpkg` is an R package meticulously crafted to provide a seamless and efficient way to interact with Ontario Hockey League data. It takes the hassle out of manual data collection by programmatically retrieving data from official OHL sources, sanitizing it, and presenting it in a clean, organized format that is ready for statistical analysis and visualization. Whether you're a sports journalist, a data science student, or a die-hard hockey fan, `OHLpkg` is your go-to tool.

Key Features

Comprehensive Data

Access a wide range of data points including player stats, team summaries, game events, and schedules for multiple seasons.

Lightning Fast

Efficiently retrieves data with minimal latency, allowing you to quickly get the information you need for real-time analysis.

User-Friendly

The retrieved data is automatically cleaned and formatted into R data frames, making it immediately usable without extra effort.

How It Works

`OHLpkg` acts as a powerful bridge, connecting your R environment directly to the official OHL data sources. It handles all the complex API calls, data scraping, and parsing in the background, so you can focus on what matters: exploring the data.

With just a single function call, the package returns a meticulously cleaned and structured data frame. This process saves you countless hours of manual work and ensures your analysis is built on a reliable foundation of high-quality data.

  • Request: You call a function like `get_RawStats()`.
  • Processing: The package fetches data from OHL servers.
  • Output: A clean data frame is returned directly to your R environment.

Installation

To install `OHLpkg`, you need the `devtools` package installed in R. If you do not have it, install it first. 


# Make sure devtools is installed
install.packages("devtools")

# Install the OHLpkg package directly from GitHub
devtools::install_github("NoahCornish/OHLpkg")

# Load the package into your R session
library(OHLpkg)

Usage Examples

Get started with these common functions to retrieve the OHL data you need.

Retrieve Raw Statistics

Use the `get_RawStats()` function to get a comprehensive data frame of player and team statistics for a specific season. If no season is specified, it defaults to the most recent completed season. 


# Retrieve statistics for the 2025-2026 Regular Season (default)
df_current_season <- get_RawStats()

# Get data for a previous season, for example, 2023-2024
df_previous_season <- get_RawStats(season_name = "2023-2024 Season")

# You can also specify other types of seasons
df_playoffs <- get_RawStats(season_name = "2025-2026 Playoffs")

Retrieve Game Events

The `get_GameEvents()` function allows you to drill down into a single game, retrieving a play-by-play log of all events. 


# Retrieve detailed events for a specific game ID
game_events <- get_GameEvents(game_id = 12345)

# Example of what the data frame looks like
head(game_events)
#   GameID Period Event Time AwayTeam HomeTeam Scorer Assist1 ...
# 1 12345  1      Goal     12:30 Away     Home     PlayerA PlayerB ...
# 2 12345  1      Penalty  15:00 Away     Home     ...
